# Sequência de comandos utilizados:

## git branch feature/header - Cria uma branch feature/header;

## git branch feature/footer - Cria uma branch feature/footer;

## git branch feature/sidebar - Cria uma branch feature/sidebar;

## git branch - Informa as branch que estão configuradas;

    feature/cadastro
    feature/calculadora
    feature/footer
    feature/header
    feature/login
    feature/sidebar
    * main
    next
    sprint

## git checkout feature/header - Muda para a branch feature/header;

## touch header.txt - Cria o arquivo header.txt;

## git add header.txt - Adiciona o header.txt à stage;

## git commit -m"feature/header: criação da documentação do header" - Versiona o arquivo da stage;

## git push -u origin feature/header - Envia o PR para o repositório remoto;

## PR - GitHud:

### Title: feature/header: criação da documentação do header;

### Description: Criação da documentação do header;

## touch sidebar.txt - Cria o arquivo header.txt;
